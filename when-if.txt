＊ Further Study ：なぜ、時・条件の副詞節中には現在形を使うのか？
　諸説ありますが、英語史を整理して考えてみると若干の流れが見えてきます。
元々、時・条件の副詞節には、仮定法現在を使い動詞の原形で表していました。
Here will I stand till Caesar pass along.（Shakespeare )
　仮定法を使った理由は、仮定法の定義 -「ある事柄を事実としてではなく、現実に反する仮定的なものとして、あるいは、現実には存在せず話し手の頭の中だけで考えられているものとして表現する方法」を考えれば明らかで、未来のことは現実には存在しないことだからだと思われます。ということで、元々時・条件の副詞節には未来形は使われていなかったということには注目しておくべきだと思います。それが変化して、仮定法現在つまり動詞の原形に最も形態的に近い現在形が使われるようになったのではないかと思われます。

まず最初に、「副詞節」とは何か、について考えてみましょう。
副詞節とは、副詞の働きをする節 （＝文の一部でＳ＋Ｖを備えたもの） で、具体的には、if 「もしも ～　ならば」とか when 「～ するとき」などの従属接続詞（従属節を主節に結びつける接続詞、注１) によって導かれる節のことで、主として主節 （注２) の動詞を修飾する副詞の働きをします。

（注１）従属接続詞によって導かれる節は、副詞の働きをする「副詞節」だけでなく、名詞の働きをする「名詞節」もありますが、ここでは「副詞節」のみを取り上げています（名詞節の説明や副詞節との区別は後述します）。
従属接続詞とは、等位接続詞（文法上対等の関係にある語と語、句と句、節と節を結びつける語で、and / but / or / nor / so /  for  があります）以外の多くの接続詞と考えてください。
（注２）主節とは、従属接続詞を含まない方の節を指します。
例えば、When  you get regular excise, you'll keep in shape. やYou'll keep in shape if you get regular exercise. の下線部の赤字が主節です。

次に、「時・条件」について考えてみましょう。
この２つに決定的な大きな違い（差）はあるのでしょうか？
「定期的に運動をすると健康が保てます」の「定期的に運動をすると」は、「時」でしょうか、それとも「条件」でしょうか？
「定期的に運動をすると」を「定期的に運動をする時」と考えれば「時」、「もし定期的に運動をするならば」と考えれば「条件」と解釈され、両者に決定的な大きな違い（差）がないことが理解できますね。
英語で書くと、下記のように２通りの英語ができると思います。
When  you get regular excise, you'll keep in shape.
If you get regular excercise, you'll keep in shape.
【注意】ここで、when と if の使用に関しての注意点があります。
それは、上記の例文のように「繰り返し起こること」について言う場合は、when と if はほとんど区別をしなくても用いられますが、次の例のように、物事が起こることがはっきりしていない場合」は、when を用いるのは誤りで、if になります。
We will have a barbecue in the yard this evening
if 「（×） when ]  it does not rain.
「雨が降らなければ今夜は庭でバーベキュー・パーティをします」

「時」と「条件」は厳密に言えば上記のように微妙に違う点もありますが、両者に決定的な大きな違い（差）がないことが理解できますね。
分かりやすく説明するために「(前提）条件」だけに絞って考えてみることにします。
まず、「(前提) 条件」とは一体どんなものでなくてはならないでしょうか？
言い換えれば、「(前提) 条件」はどうあるべきものでしょうか？
(前提）条件に「不確実な [不確定な] もの」があれば、それは果たして「（前提) 条件」としての役割（機能）を十分果たしうるでしょうか？

例えば、「明日雨が降るなら、私たちの遠足は延期になります」と言う場合、
私たちの遠足が 100 ％ 延期になる「(前提) 条件」とは？
そうです。「明日雨が降る（なら）」ということですね。
この (前提）条件に will という「～であろう（推量）」を表わす助動詞を入れることが果たしてできるのでしょうか？

日本語でも 「明日雨が降るであろうなら」と は言いませんね。
「明日雨が降る（なら）」ときっぱりと言い切らなければ、絶対に「(前提) 条件」にはなり得ません。
ですから、条件を表わす副詞節の if  節に「～であろう」を表わす「確実ではない推量」の will を入れることが出来ないのです。
 
《  「総合英語 Forest（桐原書店）」 p.74 「時や条件を表す接続詞の後で用いる現在形」 》
when や if などの「時」や「条件」を表す接続詞を用いると、それに続く節の中では
「実際に成り立つ」と考えられることを扱うことになるという記述があります。
ですから、上記の「雨が降る」のが「実際にあること」だとしたらどうなるのかを考えて、その場合は、「私たちの遠足は延期になります」と言っている訳ですね。
The birds will fly south when winter comes. 
「冬が来ると、その鳥は南へ飛んで行くだろう」でも「冬が来る」のを実際にあることと考えて、「実際に冬が来た場合には」、「鳥が南へ飛んで行くだろう」というように主節の内容とつないでいるのです。
Forest の説明は実に明快です。上記の「実際にあること」は未来表現のところで見たような「予測」と関係がないので、 接続詞の when や if などの後では、will や be going to を用いず、動詞は単純な現在形にすると記述されています。

「明日雨が降るなら、私たちの遠足は延期になります」は、If it will rain tomorrow, ではなく、
If it rains tomorrow, と「現在形」で言い切ってください。
If it rains tomorrow, our picnic will be put off. と英訳されます。
『「時」や「条件」を「決まったこと、前提」として，言うので,現在形を用いる』や『 英文の主節に will を用いているので、条件節　（ If  節 ) に will を省く』といういわば「省エネ理論」を主張される方もいます。

このことに関しては、「英文法解説（金子書房）」 p. 212 に以下のような記述があります。
『時や条件の副詞節で現在時制が使われる現象に対する伝統的な英文法の考え方は、「主節の内容が未来を表しているから、誤解のない限り副詞節まで未来を表す will を使う必要はない」という簡単な理論に基づいている。つまり言葉の省力化 (economy of speech ）であるが、名詞節の場合は時の前後関係に誤解が生じる恐れがあるので、副詞節と同一には扱えない。』
また、同書 p. 240 に 「(時・条件の）副詞節における現在完了」に関する以下の記述があります。『副詞節における現在完了は、意味の上では未来完了に相当する』

例文
* This work will come easy for him when he has had more experience.
「もっと経験を積めば、この仕事は彼にも簡単にできるようになるだろう」
has had は will have had の意だが、will　をつけてはいけない。
* Give your hair a good rinse after you've shampooed it.
「髪を洗ったら、後でよくリンスをしなさい」
have shampooed は will have shampooed の意だが、will をつけてはいけない。

このように、「時・条件」を表す副詞節では will は使えなく、現在形や現在完了形になります。これは「文法の時制に関する重要なルール」ですので、よく理解する必要があると思います。
上記で「条件」≒「時」というこことを確認しましたので、「時」を表す副詞節に関する記述は不要でしょう。「時」を表す副詞節は 接続詞 when  で始まるものだけでなく、他にもいくつかの「時」を表す接続詞で始まるものがあります。
具体例を少し「ロイヤル英文法（旺文社）」などから拾ってみましょう。
When the moie is over, I'm going straight home.
「映画が終わったら、まっすぐ家に帰ります」 （is は will be にしないこと）
Wait until I call. 「私が呼ぶまで待ちなさい」 （call は will call にしないこと）
As soon as you  (× will )  arrive in Kobe, please let me know.
「神戸に着いたらすぐ知らせてください」 (「Genius 英和辞典（大修館書店）」）

ここからは「応用編」です。少しレベルがアップします。
I don't know などで始まる英文のあとに 疑問詞 when  「 いつ ～」や 接続詞 if  「～ かどうか」で始まる間接疑問 (注３）が続くときは、when や if  で始まる節中であっても  will は使えるのです！　
例えば、I don't know when [if] he will come.
「彼がいつ来るか [彼が来るかどうか] は、私にはわかりません」
なぜでしょうか？
これらは「副詞節」ではなく、主節の動詞の「目的語」になっている「名詞節」だからです。
上の英文では、don't know の目的語が when [if] 以下の名詞節です。
主節の動詞の目的語になる「名詞節」と主節の動詞や主節全体を修飾する「副詞節」との区別はとても重要です。
when や if で始まる節が名詞節か副詞節かは日本語訳でも簡単に分かります。
名詞節の if ：「 ～ かどうか」、副詞節の if ：「もしも ～ ならば」
名詞節の when ：「いつ ～ するか」、副詞節の when 「～ するとき」
《補足》 「名詞」と「副詞」の区別は大丈夫ですね。
名詞は文中では、「主語 / 目的語 / 補語」になることができます（副詞は不可）。
副詞は、主として「動詞」を修飾しますが、その他に形容詞、副詞、文全体を修飾することもあります（名詞は不可）。
名詞節と副詞節の区別もこれと同じようなことが言えるのです。 
「副詞節」は、主として「主節の動詞を修飾する節」と覚えましょう。
名詞や副詞の働きは、基本的には、名詞節や副詞節になっても変わりません
「節（＝文の一部でＳＶを備えているもの）」。
《注３》 間接疑問とは、「疑問詞 または if [whether] ＋Ｓ＋Ｖ」の形をしたものです。

副詞節という難しい文法用語を用いずに、「時や条件を表す接続詞の後では、
現在形を用いる」という表現を使った方が分かりやすかったかも知れません。
また、副詞節の when ～ は「～ するとき」、if ～ は「もしも ～ ならば」と訳し、
名詞節の when ～ は「いつ ～ 」、名詞節の if  ～ は「～ かどうか」と訳す、
とした方が分かりやすかったかもしれません。
名詞節・形容詞節（= 関係詞節）・副詞節に見られる「節」を知らずに文法の学習は
不可能だと思い、あえて「節」の説明に多くのスペースを割きました。
かえって分かりにくいというご批判を承知しています。
「時・条件を表わす副詞節 の中では（時や条件を表す接続詞の後では）、
未来の事柄を表すのに現在形を用いる」ことの理解は深まりましたでしょうか？
will は本来「予測」を表すので、時・条件の副詞節は「予測」を表わすのではないからです。
今回の改訂版では「(時・条件の）副詞節における現在完了は、意味の上では未来完了に相当する」を追加しました。
上記の文法ルールを理解するだけでは不十分だと思います。
そのことを特に意識しなくても、現在形や現在完了形が使えるレベルまで到達しましょう。
そのためには、文脈のある生きた英文の中で出来る限り多くの実例に接したり、
その形を用いた模範的な例文を音読・暗唱することなどが必要でしょう。
input した文法を 上記のような方法で intake しない限り output できないと思います。
